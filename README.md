# Calculadora React + Vite

Calculadora básica desarrollada como práctica de React y Vite.

## Características
- Operaciones básicas (+, -, *, /)
- Diseño responsive
- Interfaz moderna

## Tecnologías
- React
- Vite
- CSS

## Instalación

\`\`\`bash
npm install
npm run dev
\`\`\`
